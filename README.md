# KhSol-Nest

­Ъџђ **KhSol-Nest** рЃљрЃарЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃБрЃарЃў RESTful API рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ, рЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў [NestJS](https://nestjs.com/)-рЃќрЃћ рЃЊрЃљ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў MongoDB-рЃЌрЃљрЃю.

---

## ­ЪЊд рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

- **NestJS** РђЊ TypeScript-рЃќрЃћ рЃљрЃњрЃћрЃЉрЃБрЃџрЃў scalable рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃБрЃарЃў рЃЕрЃљрЃарЃЕрЃЮ
- **MongoDB + Mongoose** РђЊ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
- **JWT Authentication** РђЊ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў
- **RBAC (Role-based Access Control)** РђЊ granular-level рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў
- **Swagger** РђЊ API-рЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
- **Class-validator & class-transformer** РђЊ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- **Nodemailer + Handlebars** РђЊ Email рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ
- **Docker-ready рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ** РђЊ рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃљ

## РџЎ№ИЈ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

### 1. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ

```bash
git clone https://github.com/your-username/khsol-nest.git
cd khsol-nest
```

### 2. рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

```bash
npm install
```

### 3. .env рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

`.env` рЃцрЃљрЃўрЃџрЃў (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў):

```env
PORT=3000
MONGO_URI=mongodb://localhost:27017/khsol-db
JWT_SECRET=your_jwt_secret
JWT_EXPIRES_IN=1d
SMTP_USER=your_email@gmail.com
SMTP_PASSWORD=your_email_password
```

### 4. рЃљрЃърЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

```bash
npm run start:dev
```

## ­Ъџђ API Endpoint-рЃћрЃЉрЃў

Swagger UI рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ:

```
http://localhost:3000/api
```

> рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљ endpoint-рЃА, DTO-рЃА рЃЊрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃЌ.

## ­ЪЊЂ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

| рЃЏрЃЮрЃЊрЃБрЃџрЃў       | рЃљрЃдрЃгрЃћрЃарЃљ |
|--------------|---------|
| `auth`       | рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ, рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, token рЃЏрЃљрЃарЃЌрЃЋрЃљ |
| `users`      | рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА CRUD |
| `roles`      | рЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ |
| `profile`    | рЃўрЃБрЃќрЃћрЃарЃўрЃА рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў |
| `basket`     | рЃЎрЃљрЃџрЃљрЃЌрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў |
| `procedures` | рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў |
| `token`      | refresh/access рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃфрЃљрЃџрЃЎрЃћ рЃЏрЃЮрЃЊрЃБрЃџрЃў |
| `mail`       | рЃЏрЃћрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ рЃЊрЃљ рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ |
| `common`     | shared рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў (pipes, guards, dto-рЃћрЃЉрЃў) |

## ­Ъљ│ Docker (рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў)

### Dockerfile

```Dockerfile
FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
CMD [ "node", "dist/main.js" ]
```

### docker-compose.yml (MongoDB-рЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ)

```yaml
version: '3.8'
services:
  api:
    build: .
    ports:
      - "3000:3000"
    env_file:
      - .env
    depends_on:
      - mongo

  mongo:
    image: mongo:5
    ports:
      - "27017:27017"
    volumes:
      - mongo_data:/data/db

volumes:
  mongo_data:
```
